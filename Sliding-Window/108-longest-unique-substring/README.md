## longest unique substring

Write a function that takes in a string as an argument. The function should return the length of the longest substring
that consists of only unique characters.

#### test_00:

```python
longest_unique_substring("abcabcqbb") # -> 4
# 'abcq' is the longest substring with unique characters and its length is 4
```

#### test_01:

```python
longest_unique_substring("forgeonwards") # -> 9
```

#### test_02:

```python
longest_unique_substring("tmmzuxt") # -> 5
```

#### test_03:

```python
longest_unique_substring("racecarisgreatness") # -> 8
```

#### test_04:

```python
longest_unique_substring("zzzzzz") # -> 1
```

#### test_05:

```python
longest_unique_substring("zqzzzz") # -> 2
```
