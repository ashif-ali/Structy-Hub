## longest two char substring

Write a function that takes in a string as an argument. The function should return the length of the longest substring
that consists of 2 distinct characters

#### test_00:

```python
longest_two_char_substring("xyzyyx") # -> 4
# 'yzyy' is the longest substring of 2 distinct characters and its length is 4
```

#### test_01:

```python
longest_two_char_substring("qrss") # -> 3
```

#### test_02:

```python
longest_two_char_substring("ababba") # -> 6
```

#### test_03:

```python
longest_two_char_substring("ttttttt") # -> 0
```

#### test_04:

```python
longest_two_char_substring("abacbbcbxadd") # -> 5
```
